Задача Два велосипедиста
Имя входного файла: стандартный ввод
Имя выходного файла: стандартный вывод
Ограничение по времени: 1 с
Ограничение по памяти: 256 МБ
Два велосипедиста одновременно начинают ехать по кольцевому треку длины LLL в одну и ту же сторону. Их скорости постоянны и равны соответственно v1v_1v1​ и v2v_2v2​.
Определите расстояние между велосипедистами в момент, когда более медленный из них сделает NNN кругов по треку. Расстояние измеряется не по прямой, а по треку, поскольку он не обязательно имеет форму круга.

Формат входных данных

Единственная строка содержит четыре целых числа L,v1,v2,NL, v_1, v_2, NL,v1​,v2​,N (1≤L,N≤10001 \le L, N \le 10001≤L,N≤1000, 1≤v1,v2≤5001 \le v_1, v_2 \le 5001≤v1​,v2​≤500, v1≠v2v_1 \neq v_2v1​=v2​).

Формат выходных данных

Выведите единственное число — искомое расстояние, вычисленное с точностью до 10−410^{-4}10−4.

стандартный вводстандартный вывод
10 12 7 5 4.2857
197 28 7 65 0.0000
