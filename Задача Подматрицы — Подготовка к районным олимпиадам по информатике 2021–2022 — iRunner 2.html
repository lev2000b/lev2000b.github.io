Пусть задана целочисленная матрица A из MMM строк и NNN столбцов. Назовём подматрицей этой матрицы прямоугольную часть исходной матрицы, начиная с ячейки (i1,j1i_1, j_1i1​,j1​) и заканчивая ячейкой (i2,j2i_2, j_2i2​,j2​).
Вам требуется выполнить QQQ запросов на нахождение суммы элементов в различных подматрицах матрицы A.

Формат входных данных

Первая строка содержит величины M,N,QM, N, QM,N,Q (1≤M,N,Q≤100 0001 \le M, N, Q \le 100\,0001≤M,N,Q≤100000, N⋅M≤1 000 000N \cdot M \le 1\,000\,000N⋅M≤1000000). Далее следуют MMM строк из NNN чисел каждая — элементы матрицы A. Модули значений элементов не превосходят 10610^6106.
После этого следуют QQQ строк , каждая из которых содержит параметры одного запроса — величины i1,j1,i2,j2i_1, j_1, i_2, j_2i1​,j1​,i2​,j2​ (1≤i1≤i2≤M,1≤j1≤j2≤N1 \le i_1 \le i_2 \le M, 1 \le j_1 \le j_2 \le N1≤i1​≤i2​≤M,1≤j1​≤j2​≤N).
Дополнительные ограничения:
  • В 20% тестов 2≤M,N,Q≤1002 \le M, N, Q \le 1002≤M,N,Q≤100.
  • В 30% тестов M=1M=1M=1.

Формат выходных данных

Выведите в одну строку QQQ чисел — результаты выполнения запросов. Порядок следования этих чисел должен соответствовать порядку запросов во входном файле.
